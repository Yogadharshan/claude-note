## definition

A language L is **Context-Free** (CFL) if there exists a Context-Free Grammar (CFG) G such that L = L(G).

Equivalently: L is CFL ↔ L is recognized by some Pushdown Automaton (PDA).

## intuition

CFLs are exactly the languages that can be described by recursive rules with one level of "counting" or nesting. They can handle balanced structures (a^n b^n) but not independently counting three things simultaneously (a^n b^n c^n).

## key examples

**IN CFL:**
- {a^n b^n | n ≥ 0} — equal a's then b's
- {w | w is a palindrome over {a,b}} — reversible strings
- Balanced parentheses — ()((()))
- Arithmetic expressions — a*(b+c)
- Most programming language syntax
- {a^n b^m c^n | n,m ≥ 1} — a^n c^n with any b's in between

**NOT in CFL:**
- {a^n b^n c^n | n ≥ 1} — all three counts equal
- {ww | w ∈ {0,1}*} — string repeated twice
- {a^n | n is a prime} — prime-length strings
- {a^(2^n) | n ≥ 0} — lengths that are powers of 2

## closure properties

| Operation | CFL closed? |
|-----------|-------------|
| Union | YES |
| Concatenation | YES |
| Kleene Star | YES |
| Intersection | NO |
| Complement | NO |
| Intersection with Regular | YES (important!) |

**Proof that intersection is NOT closed:**
- L1 = {a^n b^n c^m} — CFL
- L2 = {a^m b^n c^n} — CFL
- L1 ∩ L2 = {a^n b^n c^n} — NOT CFL
Therefore CFLs are not closed under intersection.

## decidable properties of CFLs

| Property | Decidable? |
|----------|-----------|
| Is w ∈ L(G)? (Membership) | YES — via PDA or CYK algorithm |
| Is L(G) = ∅? (Emptiness) | YES — check if S can derive terminal string |
| Is L(G) infinite? | YES — check for cycles in CNF grammar |
| Is L(G1) = L(G2)? (Equivalence) | NO — UNDECIDABLE |

## applications

- Programming language syntax
- XML/HTML document structure
- Mathematical expression parsing
- Natural language syntax (approximately)
- Protocol message formats

## relationship to other classes

```
Regular ⊂ CFL ⊂ Context-Sensitive ⊂ Recursively Enumerable
```
- Every regular language is also context-free (but not vice versa)
- CFL ∩ Regular = CFL (CFLs are closed under intersection with regular languages — extremely useful!)
